# Sky Moss

A minimal ambient side-game designed for a wide, narrow taskbar-like layout.

## Core Idea
The player gently manages a tiny moss garden by controlling drifting clouds.

- Move clouds
- Trigger light rain
- Let sunlight through
- Shape a living miniature ecosystem

## Gameplay
The world evolves mostly on its own:
- moss spreads
- dew forms
- snails appear
- flowers bloom
- fog gathers

The player interacts only occasionally for a few seconds at a time.

## Design Goal
A calm background game that:
- never demands attention
- feels alive while working
- rewards returning after long focus periods

## Unique Touch
The garden can subtly react to user activity:
- typing → breeze
- idle time → mist
- long sessions → rare growth
